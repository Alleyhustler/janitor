Notion

Этот раздел посвящен библиотеке для работы с Notion API в 1С:Предприятие, OneScript и CLI. На данной странице описаны все действия, необходимые для полноценного начала работы

Начало работы​

  1. Перейдите в раздел интеграций портала для разработчиков

  1. Введите название вашей интеграции

  1. После завершения создания интеграции, нажмите Show и скопируйте Токен (Secret)

  1. Перейдите в основной интерфейс Notion и создайте новую страницу. Эта страница будет родительской для всех остальных объектов, создаваемых через API

  1. Нажмите три точки в правом верхнем углу и в самом низу найдите пункт Add connections. Выберите там свою интеграцию из пунктов 1-3

  1. Получите ID этой страинцы. Это можно сделать нажав Share -> Copy link или просто скопировав из URL

Токен и ID страницы - два параметра, необходимые для работы через API
