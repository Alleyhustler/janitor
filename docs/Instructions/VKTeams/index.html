VK Teams

Этот раздел посвящен библиотеке для работы с VK Teams API в 1С:Предприятие, OneScript и CLI. На данной странице описаны все действия, необходимые для полноценного начала работы

Начало работы​

  1. Зарегистрируйтесь, оформите тариф и скачайте приложение VK Teams на главной странице

  1. Войдите в приложении при помощи одноразового пароля, который должен прийти на почту. Просмотреть почту необходимого пользователя можно в панели администрирования

  1. Начните диалог с @metabot и пройдите процедуру создания нового бота

  1. Сгенерируйте новый токен и включите возможность добавлять бота в группу (если необходимо)

  1. Сохраните ID бота, получив его при помощи кнопки "Показать моих ботов"

  1. Сформируйте ссылку и перейдите по ней в браузере

На конце URL должен быть ваш ID бота
https://teams.vk.com/profile/1234567890

  1. Начните диалог с ботом

Теперь вы сможете добавлять бота в группы и управлять им при помощи Токена и ID необходимого диалога (группы или пользователя)
