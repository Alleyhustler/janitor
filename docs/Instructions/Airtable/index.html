Airtable

Этот раздел посвящен библиотеке для работы с Airtable API в 1С:Предприятие, OneScript и CLI. На данной странице описаны все действия, необходимые для полноценного начала работы

Начало работы​

  1. Авторизуйтесь в Airtable и перейдите на страницу создания персональных токенов, нажмите Create new token

  1. Введите название для нового токена, выберите из перечня необходимые Scopes (разрешения) и Access (базы и рабочие пространства, к которым у токена будет доступ)

  1. Скопируйте новый токен
