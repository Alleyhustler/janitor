Slack

Этот раздел посвящен библиотеке для работы со Slack Web API в 1С:Предприятие, OneScript и CLI. На данной странице описаны все действия, необходимые для полноценного начала работы

Начало работы​

  1. Войдите в свой аккаунт и перейдите на страницу Your Apps на сайте Slack API


  1. Нажмите Create App и выберите пункт From scratch

  1. Выберите название и рабочую область, в которой необходимо создать приложение


  1. Создайте новый токен, нажав Generate Token and Scopes ниже на странице приложения


  1. Выберите имя и добавьте разрешения


  1. Перейдите в раздел Premission scope из пункта установки приложения


  1. Добавьте необходимые для ваших целей разрешения


  1. Установите приложение


  1. Сохраните сгенерированный токен
