Обновить токен

Обновляет токен по Refresh токену

Функция ОбновитьТокен(Знач ClientID, Знач ClientSecret, Знач Refresh) Экспорт

ПараметрCLI опцияТипОбяз.Назначение
ClientID--clientidСтрока✔Client ID из настроек приложения
ClientSecret--clientsecretСтрока✔Client secret из настроек приложения
Refresh--refreshСтрока✔Refresh токен

Возвращаемое значение: Соответствие Из КлючИЗначение - сериализованный JSON ответа от Bitrxi24


Пример использования для 1С:Предприятие/OneScript
    ClientID     = "local.6667fc928a50a9.70414732";
    ClientSecret = "ZeKyeYIgy2NsHZqsIHY6GfG1V...";
    Refresh      = "3d6d5468006e9f06006b12e400000001000...";

    Результат = OPI_Bitrix24.ОбновитьТокен(ClientID, ClientSecret, Refresh);
  • Bash
  • CMD/Bat

      --clientid "***" \
      --clientsecret "***" \
      --refresh "***"
    oint bitrix24 ОбновитьТокен ^
      --clientid "***" ^
      --clientsecret "***" ^
      --refresh "***"
