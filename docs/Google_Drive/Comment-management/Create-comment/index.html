Создать комментарий

Создает комментарий к файлу или каталогу

Функция СоздатьКомментарий(Знач Токен, Знач Идентификатор, Знач Комментарий) Экспорт

ПараметрCLI опцияТипОбяз.Назначение
Токен--tokenСтрока✔Токен
Идентификатор--objectСтрока✔Идентификатор объекта, для которого необходим комментарий
Комментарий--textСтрока✔Текст комментария

Возвращаемое значение: Соответствие Из КлючИЗначение - сериализованный JSON ответа от Google


Пример использования для 1С:Предприятие/OneScript
    Токен         = "ya29.a0AW4XtxhtghlL7aK7VhJX0jOgNWau4CUoOEDkQBr4uJEjPidCjKm-4r4l7AoJnODBt7yblowJtvv_3vETd6CulW4V5zxxPx8TTvQAMESj7...";
    Идентификатор = "1jnYtECZqTZpHN3SBvNxTvytyYwAk6q8d";
    Комментарий   = "Текст комментария";

    Результат = OPI_GoogleDrive.СоздатьКомментарий(Токен, Иденти фикатор, Комментарий);
  • Bash
  • CMD/Bat
    oint gdrive СоздатьКомментарий \
