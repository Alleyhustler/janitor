Получить список каталогов

Получает список каталогов диска

Функция ПолучитьСписокКаталогов(Знач Токен, Знач ИмяСодержит = "", Знач Подробно = Ложь) Экспорт

ПараметрCLI опцияТипОбяз.Назначение
Токен--tokenСтрока✔Токен
ИмяСодержит--querryСтрока✖Отбор по имени
Подробно--depthБулево✖Добавляет список файлов к полям каталога

Возвращаемое значение: Соответствие Из КлючИЗначение - Массив соответствий каталогов


Пример использования для 1С:Предприятие/OneScript
    Имя   = "Тестовая папка";
    Токен = "ya29.a0AW4XtxhtghlL7aK7VhJX0jOgNWau4CUoOEDkQBr4uJEjPidCjKm-4r4l7AoJnODBt7yblowJtvv_3vETd6CulW4V5zxxPx8TTvQAMESj7...";

    Результат = OPI_GoogleDrive.ПолучитьСписокКаталогов(Токен, Имя, Истина);
  • Bash
  • CMD/Bat
    oint gdrive ПолучитьСписокКаталогов \
      --token "***" \
      --querry "Тестовая папка"
    oint gdrive ПолучитьСписокКаталогов ^
      --token "***" ^
      --querry "Тестовая папка"
