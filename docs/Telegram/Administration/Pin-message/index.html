Закрепить сообщение

Закрепляет сообщение в шапке чата

Функция ЗакрепитьСообщение(Знач Токен, Знач IDЧата, Знач IDСообщения) Экспорт

ПараметрCLI опцияТипОбяз.Назначение
Токен--tokenСтрока✔Токен
IDЧата--chatСтрока, Число✔ID целевого чата
IDСообщения--messageСтрока, Число✔ID целевого сообщения

Возвращаемое значение: Соответствие Из КлючИЗначение - сериализованный JSON ответа от Telegram


подсказка

Метод в документации API: pinChatMessage


Пример использования для 1С:Предприятие/OneScript
    Токен       = "6129457865:AAFyzNYOAFbu...";
    IDКанала    = "@testsichee";
    IDСообщения = "17253";

    Результат = OPI_Telegram.ЗакрепитьСообщение(Токен, IDКанала, IDСообщения);
  • Bash
  • CMD/Bat
